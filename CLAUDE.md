# LAFC Content Scoreboard

## What this is
Analyzes LAFC's YouTube content performance against the MLS match
calendar to find what drives fan engagement. Portfolio project.

## Stack
- Python for data pulls (YouTube Data API)
- SQLite for storage; SQL queries live in /sql
- Vanilla JS/HTML dashboard

## How I work
- Explain analytical choices in plain language — I'm learning.
- Keep functions small and commented.
- Never print API keys or secrets to cell/terminal output.
- Secrets go in .env (gitignored); never hardcode them.
